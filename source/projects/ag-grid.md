ag-Grid is a framework agnostic feature-rich datagrid. 
This overview details all stand out features in ag-Grid 
from Grouping, Pivoting, Pagination, Tree Data and many more.

#### Features Overview

* Sorting
* Filtering
* Resizing
* Pinning
* Selection
* Value Getters
* Cell Styling
* Cell Rendering
* Cell Templates
* Editing Cells
* Context
* Refresh
* Grouping Headers
* Header Rendering
